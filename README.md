# Smeta Qt 2

Проект для работы со сметами в двух интерфейсах:

- десктопное приложение на PyQt5;
- веб-интерфейс на FastAPI.

Общий смысл проекта: загрузить смету, обработать её, сопоставить позиции, сравнить две сметы и выгрузить результат в удобном виде.

## Что умеет проект

- обработка смет из Excel-файлов;
- сравнение двух смет с выводом различий;
- работа с материалами и привязками материалов к данным сметы;
- экспорт результатов в Excel и HTML;
- веб-версия для тех же сценариев обработки и сравнения;
- вспомогательное форматирование таблиц для более читаемого вывода.

## Основные части кода

- `main.py` - главное окно десктопного приложения и UI-логика.
- `data_processing.py` - обработка сметных данных.
- `smeta_compare.py` - сравнение двух смет и сбор отчётов.
- `export_formatting.py` - оформление таблиц, HTML и Excel-вывода.
- `fact_export.py` - экспорт с формулами для фактических данных.
- `db.py` - модели и работа с SQLite-базой `smeta.db`.
- `web_app/backend.py` - FastAPI-бэкенд для веб-версии.
- `web_app/templates` и `web_app/static` - шаблоны и фронтенд веб-приложения.

## Десктопное приложение

Запуск:

```bash
python main.py
```

В коде десктопной части есть:

- главное окно приложения;
- загрузка и отображение таблиц;
- диалоги выбора колонок;
- окна для сравнения смет;
- экспорт и открытие сохранённых файлов;
- вспомогательные диалоги по недостающим данным и материалам.

## Веб-приложение

Веб-часть построена на FastAPI и использует ту же бизнес-логику, что и десктопная версия.

Типовые сценарии:

- загрузка и обработка одной или нескольких смет;
- сравнение двух файлов;
- экспорт отчётов;
- работа с материалами через API.

Подробная схема веб-части уже вынесена отдельно в `web_app/ARCHITECTURE.md`.
